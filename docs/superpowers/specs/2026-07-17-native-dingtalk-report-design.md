# 钉钉原生 Markdown 法务资讯报告设计

## 目标

每轮正式采集只推送一条钉钉 Markdown 消息。产品价值只由信息质量、内容层次、可读性和执行价值决定，不再依赖长图、网页或外部存储。

## 内容准入

- 中国信息优先，同一报告正文不得重复。
- 不强求固定数量；只有达到质量门槛的信息才进入正文。
- 行业新闻只有具备明确法务关注价值和下一观察信号时，才以“持续观察”进入。
- 六大板块结构保留，但无合格内容的板块不展示、不填充占位内容。

## 消息结构

1. 标题：`美妆法务资讯｜日期`。
2. 管理层摘要：最多 3 条，直接说明本期最重要的判断。
3. 资讯正文：按六大板块排列，每条固定包含：
   - 标题及风险/事项类型；
   - 核心判断；
   - 对业务或品牌的影响；
   - 法务动作；
   - 责任岗位，完成时间统一为“由责任领导确定”；
   - 可点击的原文来源和发布日期。
   - 每个字段名称使用一级无序列表，字段内的事实、判断、动作、岗位和来源分别使用二级无序列表；不得用分号把多个信息点挤在同一行。
4. 持续观察事项用“关注价值”和“下一观察点”替代强制行动。
5. 结尾保留简短综合判断及内部研判免责声明。

## 交付与降级

- 始终构建完整原生 Markdown，不生成或引用资讯长图。
- 单次只发送一条消息，不拆分卡片。
- 超出钉钉字节预算时，依次压缩低优先级、非中国事项；仍超限时才省略最低优先级事项，并在正文中明确说明。
- 紧凑模式保持相同两级结构，但每个字段只保留最重要的一至两项。
- 不允许发送空消息；无准入事项时发送一条明确的“本期无重大更新”简报。
- 钉钉发送失败沿用既有有限重试和明确失败返回。

## 验收标准

- 消息中不包含资讯长图或“查看高清原图”。
- 正文文字可复制，每条合格来源可直接点击。
- 中国事项排序在前；同一报告无重复事项。
- 仅一条钉钉消息，且不超过配置的字节上限。
- 有内容、无内容、超长内容和发送失败场景均有自动化测试。
